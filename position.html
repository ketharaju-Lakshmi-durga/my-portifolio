Positioning of elements in CSS

The position property describes the type of positioning method used for an element

There exist 5 different position values

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

In this blog lets see about the absolute and relative position

  • Relative position
  • Absolute position

Relative positioning

Here the elements are postioned relative to it's position in normal document flow

Positioning css(offset)properties are:

  • top
  • bottom
  • left
  • right

Remaining elements are uneffected ,they are not taken out of normal document flow unlike positioning elements like floating

realtive positioning offsets the elements relative to its normal documenet flow position

Static positioning:

Normal document flow default for all ,except for html tag

Absolute positioning

All offsets are realtive to the position of the nearest ancestor which has positioning set on it,other than static

Positioning css(offset)properties are:

  • top
  • bottom
  • left
  • right

static positioning is default for all elements ,except for html.These are not effected by offsets

Absolute positoining is relative to the closeset ancestor which has positioning set to non-static value

Note:

By default html is the only element that has non-static positioning set on it
